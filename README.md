
# testboard-e2e-tests

E2E тесты на Playwright для Testboard.

## Установка и настройка

### 1. Установите зависимости
```bash
npm install
```

### 2. Установите браузеры Playwright
```bash
npx playwright install
```

### 3. Настройте переменные окружения
Создайте в корне проекта файл `.env` и укажите следующие переменные:

```ini
E2E_USER_EMAIL=ваши@данные.ru
E2E_USER_PASSWORD=Вашпароль123
BASE_URL = https://testboard.avito.com/
```

> **Важно:** Email и пароль нужно указывать от существующего аккаунта.

## Запуск тестов

### Запустить все тесты
```bash
npx playwright test
```

### Запустить конкретный тестовый файл
```bash
npx playwright test login.spec.ts
```

### Запустить тест по названию
```bash
npx playwright test -g "Вход существующего пользователя"
```

### Дополнительные параметры запуска
- `headed` – открыть браузер в видимом режиме.
- `slowmo=1000` – замедлить выполнение шагов на 1000 мс (удобно для отладки).
- `viewport` - настройка расрешения 
Все это можно настроить в файле playwright.config.ts

Пример:
```bash
npx playwright test tests/ui/createAd.spec.ts --headed --slowmo=500
```

## Линтер (ESLint)

Проверить код на соответствие стилю:
```bash
npx eslint .
```

Автоматически исправить ошибки (где возможно):
```bash
npx eslint . --fix
```

Если после автоматического исправления останутся ошибки, их нужно исправить вручную.


## Примечание

- Для теста создания объявления (`createAd.spec.ts`) используется изображение `tests/fixtures/test-photo.jpg`. Убедитесь, что этот файл существует.
